Font Size Feature Test

Instructions:

  1. Open the Noet app at http://localhost:3001
  2. Create a new note or select an existing one
  3. Type some text in the editor
  4. Select the text you want to change
  5. Use the "Size" dropdown in the toolbar (next to Font Family)
  6. Choose a font size (e.g., 24px, 32px, 48px)
  7. Verify the text size changes
  8. Try selecting "Size" again to reset to default

Expected Behavior:

  • Font size dropdown should appear next to font family dropdown
  • Selected text should change size when a size is chosen
  • Font size should persist when saving the note
  • Different parts of text can have different font sizes
  • Font size should work with other formatting (bold, italic, colors)
