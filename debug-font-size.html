Font Size Issue Debugging

Steps to test:

  1. Go to the Noet app at http://localhost:3001
  2. Create or open a note
  3. Type some text
  4. Select the text
  5. Try changing the font size using the dropdown
  6. Check the browser developer tools console for any errors
  7. Check if the HTML shows inline styles being applied

Expected behavior:

When you select text and change the font size, you should see:

  • The text visually change size
  • HTML like: <span style="font-size: 18px">text</span>

Debugging tips:

  • Check if TextStyle extension is loaded
  • Verify the font size function is being called
  • Look for JavaScript errors in console
  • Inspect the generated HTML structure
